# Turn Structure

## Turn Sequence (strict order)

Each turn, run these phases globally in order. Order is part of the rules — it determines, e.g., that you fight *after* moving and split *after* fighting.

```text
1. PRODUCTION   2. GROWTH   3. DECISION   4. MOVEMENT
5. COMBAT   6. OCCUPATION   7. CONNECTIVITY   8. DIPLOMACY   9. LOG
```
